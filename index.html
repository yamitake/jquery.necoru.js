Getting Started

1. cssを読み込む
<link href="./necoru/necoru.css" rel="stylesheet" type="text/css" media="all" />

2. jsを読み込む. ※jqueryとjquery.nicoru.jsを読み込む。
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.5.0/jquery.min.js"></script>

<script type="text/javascript" src="./necoru/jquery.necoru.js"></script>

3. jsで初期化する。 ボタンを書き出します。
<span></span> <script type="text/javascript"> $(function() { $("#target").necoru(); }); </script>
